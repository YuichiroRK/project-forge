# ADR-0004
# Procedural Equipment

Estado:
Aceptado

Fecha:
2026-07-14

## Contexto

El sistema de crafting es el núcleo del proyecto.

No queremos un sistema basado en recetas.

## Decisión

Todo el equipamiento será generado mediante parámetros.

Las estadísticas dependerán de:

- Material.
- Forma.
- Calidad.
- Balance.
- Peso.
- Dimensiones.
- Acabado.

No existirán recetas obligatorias.

## Consecuencias

Cada objeto podrá ser único.

Se fomenta la creatividad.

El sistema requerirá un motor de evaluación complejo.
