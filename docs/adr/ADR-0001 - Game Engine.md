# ADR-0001
# Game Engine

Estado:
Aceptado

Fecha:
2026-07-14

## Contexto

Project Forge será un RPG/MMORPG 2D Pixel Art con soporte para PC y Android.

El motor debe facilitar:

- Desarrollo rápido
- Exportación multiplataforma
- Networking
- Pixel Art
- Comunidad activa

## Decisión

Se utilizará Godot 4 como motor principal.

El lenguaje principal será GDScript.

## Consecuencias

### Positivas

- Desarrollo rápido.
- Excelente soporte para 2D.
- Open Source.
- Exportación sencilla.
- Integración sencilla con Git.

### Negativas

- Ecosistema menor que Unity.
- Algunas librerías requieren implementación propia.

## Alternativas

- Unity
- Unreal Engine
- GameMaker
- Monogame
